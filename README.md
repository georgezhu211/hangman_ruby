# Hangman

## Description

A game of Hangman built with Ruby, playable from the command line.

## Installation

You must have ruby version 2.7.0 >=

## Usage

1. clone this repository to your local machine
2. cd into that directory then run the command

```
  ruby hangman.rb
```
